# ENVS-193DS_winter-2026_final

Repo for ENVS 193DS Final Project

# **General Information**

This repository contains the Winter 2026 final project for ENVS 193DS. The assignment includes statistical analysis, data visualization, and scientific communication using environmental datasets.

The project analyzes the relationship between visual obstruction at Mountain Plover nests and distance to prairie dog colony edges on the probability of nest site use, explores model simplification, summarizes a giant kelp frond dataset from the Santa Barbara Coastal LTER, communicates affective and exploratory visualizations, and includes a critical evaluation of statistical communication in a research report.

All analyses and figures are completed in R using a Quarto document with annotated code and written interpretations.
